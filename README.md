# LifeLine-AI 🚑

Smart Ambulance Emergency Dashboard

## Features
- **Real-time Requests System**: Live-updating emergency incidents with automated status workflows (Pending → Assigned → In Transit → Completed).
- **AI-based Ambulance Suggestion**: Intelligent unit recommendation algorithms to optimize dispatch response times.
- **Analytics Dashboard**: Comprehensive KPI tracking with animated, real-time data visualization.
- **Anti-gravity UI**: Premium, cinematic interface with floating card effects, glassmorphism, and neon glow aesthetics.
- **Universal Theme Sync**: Seamless light/dark mode transitions built with global CSS variables.

## Tech Stack
- **Frontend**: React 18, Vite, Vanilla CSS
- **State Management**: Custom hooks with module-level reactivity (Sub-Subscribe pattern)
- **Animations**: CSS Keyframes + `useTilt` interactive hooks
- **Mapping**: Leaflet.js with custom cinematic markers

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/lifeline-ai-dashboard.git
   ```

2. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
