# LifeLine AI

LifeLine AI is a modern, premium Emergency Response System designed with focus on aesthetics and user experience.

## ✨ Features

- **Premium Glassmorphism UI**: High-fidelity dark red neon aesthetic with depth and blur effects.
- **Controlled Simulation Flow**: Clean navigation from start simulation to intro animation and authentication.
- **Dynamic Background System**: Custom-designed background with 3D glass spheres and red neon grading.
- **Real-time Emergency Dashboard**: Unified view for ambulance tracking and emergency management.
- **Standardized Theme System**: Flicker-free global theme system with dark and light mode support.

## 🚀 Tech Stack

- **Frontend**: React, Javascript, Vanilla CSS
- **Styling**: Premium Glassmorphism, CSS Variables, Modern Typography (Inter)
- **Routing**: React Router with Stage Machine integration

## 🛠️ Getting Started

1.  **Installation**:
    ```bash
    cd frontend
    npm install
    ```
2.  **Run Locally**:
    ```bash
    npm run dev
    ```
